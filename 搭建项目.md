#### 1、github创建项目

在github上创建项目后，使用ssh本地拉取

git clone  ...

#### 2、init client

```
vue create client
	babel
	router (使用hash模式,即不适用history模式)

cd client 
vue add element
	fully
	no(不 overwrite scss)
	zh-cn
```

```
提交到仓库
git status  //查看状态
git add .   
git commit -m "init client"
git push  //远程提交
```

#### 3、init  server

```
mk server
npm init -y
npm install express body-parser morgan cors -S
npm install nodemon -D
```
